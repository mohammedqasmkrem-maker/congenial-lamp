# 🏔️ Abt Academy - أكاديمية أبت للغة الإنجليزية

![Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Version](https://img.shields.io/badge/Version-1.0-gold)

مشروع تعليمي تفاعلي متطور يعتمد على تقنيات **الذكاء الاصطناعي** و **Streamlit** لتعليم اللغة الإنجليزية بأسلوب حديث وفخم. يضم المشروع أكثر من **1000 كلمة** إنجليزية مترجمة ومنظمة برمجياً.

## ✨ المميزات الرئيسية (20 ميزة في تطبيق واحد)
* **🏔️ تصميم جبلي زجاجي:** واجهة مستخدم تعتمد على نظام *Glassmorphism* مع خلفيات جبلية فخمة.
* **📖 قاموس الـ 1011 كلمة:** قاعدة بيانات ضخمة مقسمة إلى صفحات ومستويات (سهل، متوسط، صعب).
* **🔊 نطق صوتي احترافي:** إمكانية سماع نطق كل كلمة بلمسة واحدة.
* **⚔️ نظام الاختبارات:** اختبار تفاعلي عشوائي يحسب النقاط (20 نقطة لكل إجابة صحيحة).
* **👤 ملف الإنجازات:** رتب ملكية تتغير بناءً على مستواك (مستكشف، فارس، إمبراطور).
* **⭐ قائمة المفضلة:** ميزة حفظ الكلمات الصعبة لمراجعتها لاحقاً.
* **🌿 غرفة الاسترخاء:** زاوية هادئة مدمجة بفيديو للدعاء والراحة النفسية.

## 🚀 كيف يعمل المشروع؟
يعمل المشروع عن طريق سحب البيانات حياً من ملف `vocab.csv` المرفق، ويقوم بمعالجتها برمجياً لتظهر في واجهة المستخدم بشكل منظم وسريع بفضل نظام "الصفحات" (Pagination).

## 🛠️ التقنيات المستخدمة
* **Python** - لغة البرمجة الأساسية.
* **Streamlit** - لبناء واجهة المستخدم التفاعلية.
* **CSS3** - للتصميم الزجاجي المخصص.
* **Requests & Re** - لمعالجة البيانات والنصوص.
